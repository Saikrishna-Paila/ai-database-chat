# AI Database Chat

Welcome to AI Database Chat - your natural language interface for querying databases!

## What can you do?

Simply ask questions about your data in plain English:

- "Show me all users registered this month"
- "What are the top 10 products by revenue?"
- "Count orders grouped by status"
- "Find documents where category is 'electronics'"

## Supported Databases

- **PostgreSQL** - Relational SQL database
- **MongoDB** - NoSQL document database

## Features

- Natural language to SQL/MongoDB query conversion
- Smart routing between databases
- Automatic visualizations
- Query explanation and safety validation
- Conversation context awareness

## Commands

- `/help` - Show help information
- `/schema` - Display database schema
- `/clear` - Clear conversation history
